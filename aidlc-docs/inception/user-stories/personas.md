# 테이블오더 서비스 - 사용자 페르소나

---

## Persona 1: 고객 (Customer)

### 프로필
| 항목 | 내용 |
|------|------|
| **이름** | 김민수 (대표 페르소나) |
| **역할** | 매장 방문 고객 |
| **연령대** | 20~60대 (폭넓은 연령층) |
| **기술 숙련도** | 초급~중급 (스마트폰/태블릿 기본 사용 가능) |
| **접근 디바이스** | 매장 내 테이블 태블릿 |

### 목표 (Goals)
- 대기 시간 없이 빠르게 주문하고 싶다
- 메뉴를 쉽게 탐색하고 원하는 메뉴를 찾고 싶다
- 주문 상태를 실시간으로 확인하고 싶다
- 추가 주문을 간편하게 하고 싶다

### 불만 사항 (Pain Points)
- 직원을 불러서 주문하는 것이 번거롭다
- 메뉴판이 복잡하거나 가격이 잘 안 보인다
- 주문이 제대로 들어갔는지 불안하다
- 추가 주문 시 다시 직원을 불러야 한다

### 사용 시나리오
1. 테이블에 앉으면 태블릿이 자동으로 메뉴 화면을 표시
2. 카테고리별로 메뉴를 탐색하고 장바구니에 담기
3. 장바구니에서 수량 조절 후 주문 확정
4. 주문 내역에서 현재 주문 상태 확인
5. 필요시 추가 주문 진행

### 관련 스토리
- US-01 ~ US-05 (고객 인증/세션)
- US-06 ~ US-10 (메뉴 조회/탐색)
- US-11 ~ US-15 (장바구니 관리)
- US-16 ~ US-20 (주문 생성)
- US-21 ~ US-24 (주문 내역 조회)

## Persona 2: 매니저 (Manager)

### 프로필
| 항목 | 내용 |
|------|------|
| **이름** | 박서연 (대표 페르소나) |
| **역할** | 매장 매니저 / 점장 |
| **연령대** | 30~50대 |
| **기술 숙련도** | 중급 (PC/태블릿 업무 활용 가능) |
| **접근 디바이스** | PC 또는 태블릿 브라우저 |

### 목표 (Goals)
- 매장 전체 주문 상황을 실시간으로 파악하고 싶다
- 메뉴를 쉽게 등록/수정/삭제하고 싶다
- 테이블 운영을 효율적으로 관리하고 싶다
- 직원 계정을 관리하고 권한을 설정하고 싶다

### 불만 사항 (Pain Points)
- 주문이 밀릴 때 전체 상황 파악이 어렵다
- 메뉴 변경 시 수작업이 많다
- 테이블 회전율을 관리하기 어렵다
- 과거 주문 내역 조회가 번거롭다

### 사용 시나리오
1. 출근 후 관리자 로그인
2. 대시보드에서 전체 테이블 주문 현황 모니터링
3. 신규 주문 확인 및 상태 변경
4. 필요시 메뉴 등록/수정/삭제
5. 테이블 이용 완료 처리 및 과거 내역 확인
6. 직원 계정 관리

### 권한 범위
- **전체 기능 접근**: 주문 모니터링, 주문 상태 변경, 주문 삭제, 테이블 이용 완료, 테이블 초기 설정, 과거 내역 조회, 메뉴 관리, 직원 계정 관리

### 관련 스토리
- US-25 ~ US-30 (관리자 인증)
- US-31 ~ US-38 (실시간 주문 모니터링)
- US-39 ~ US-47 (테이블 관리)
- US-48 ~ US-55 (메뉴 관리)

## Persona 3: 직원 (Staff)

### 프로필
| 항목 | 내용 |
|------|------|
| **이름** | 이준호 (대표 페르소나) |
| **역할** | 매장 직원 / 홀 서빙 |
| **연령대** | 20~40대 |
| **기술 숙련도** | 초급~중급 |
| **접근 디바이스** | PC 또는 태블릿 브라우저 |

### 목표 (Goals)
- 들어오는 주문을 빠르게 확인하고 처리하고 싶다
- 주문 상태를 쉽게 변경하고 싶다
- 테이블 이용 완료 처리를 빠르게 하고 싶다

### 불만 사항 (Pain Points)
- 바쁜 시간에 주문을 놓칠 수 있다
- 어떤 테이블이 어떤 주문을 했는지 파악이 어렵다
- 잘못된 주문 처리가 번거롭다

### 사용 시나리오
1. 출근 후 직원 계정으로 로그인
2. 대시보드에서 담당 테이블 주문 확인
3. 주문 상태 변경 (대기중 → 접수 → 준비중 → 완료)
4. 잘못된 주문 삭제 처리
5. 고객 퇴장 시 테이블 이용 완료 처리

### 권한 범위
- **제한된 접근**: 주문 모니터링, 주문 상태 변경, 주문 삭제, 테이블 이용 완료, 과거 내역 조회
- **접근 불가**: 테이블 초기 설정, 메뉴 관리, 직원 계정 관리

### 관련 스토리
- US-25 ~ US-30 (관리자 인증 - 직원 역할)
- US-31 ~ US-38 (실시간 주문 모니터링)
- US-39 ~ US-47 (테이블 관리 - 제한된 권한)
